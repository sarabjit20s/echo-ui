## Introduction

EchoUI is a collection of pre-built, high-quality React Native components designed to accelerate your mobile app development. Components are designed to deliver smooth and efficient performance on both Android and iOS. All components are made with [`react-native-unistyles`](https://www.unistyl.es/).

## Installation

<Steps>

<Step>
### Install Unistyles
For more information, see [Unistyles](https://www.unistyl.es/).
</Step>

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn']}>
```bash tab="npm"
npm install react-native-unistyles
```

```bash tab="pnpm"
pnpm add react-native-unistyles
```

```bash tab="yarn"
yarn add react-native-unistyles
```
</Tabs>

<Step>
### Setup Unistyles and Theming

1. Create a `styles` directory in your project. This is where you will keep your all themes, tokens or design system.
2. Create a `tokens` directory in your project. This is where you will keep your all tokens.
3. You need to create `colors.ts`, `radius.ts`, `space.ts` and `typography.ts` files in your `tokens` directory.

4. Add the following code to your respective tokens files.
<Callout title="Typography">Download [Inter](https://rsms.me/inter) font and follow this [docs](https://docs.expo.dev/develop/user-interface/fonts/#use-a-local-font-file) if you are using Expo to use it in your project.</Callout>

5. Create a `themes.ts` file in your `styles` directory and add the following code:
```ts
import { lightThemeColors, darkThemeColors } from './tokens/colors';
import { radius } from './tokens/radius';
import { space } from './tokens/space';
import { typography } from './tokens/typography';

export const lightTheme = {
  colors: lightThemeColors,
  radius,
  space,
  typography,
} as const;

export const darkTheme = {
  colors: darkThemeColors,
  radius,
  space,
  typography,
} as const;

export const themes = {
  light: lightTheme,
  dark: darkTheme,
} as const;

// every theme should have a same structure as lightTheme
export type Theme = typeof lightTheme;
export type Themes = typeof themes;

```

6. Create a `breakpoints.ts` file in your `styles` directory and add the following code:
```ts
export const breakpoints = {
  mobile: 0,
  tablet: 768,
} as const;

export type Breakpoint = keyof typeof breakpoints;
export type Breakpoints = typeof breakpoints;

```

7. Create a `unistyles.ts` file in your `styles` directory and add the following code:
```ts
/* eslint-disable @typescript-eslint/no-empty-object-type */
import { UnistylesRegistry } from 'react-native-unistyles';

import { Breakpoints, breakpoints } from './breakpoints';
import { themes, Themes } from './themes';

declare module 'react-native-unistyles' {
  export interface UnistylesBreakpoints extends Breakpoints {}
  export interface UnistylesThemes extends Themes {}
}

UnistylesRegistry.addBreakpoints(breakpoints).addThemes(themes).addConfig({
  adaptiveThemes: true,
});

```
