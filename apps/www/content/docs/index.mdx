## Introduction

**EchoUI** is a collection of pre-built, high-quality React Native components designed to accelerate your mobile app development. Components are designed to deliver smooth and efficient performance on both Android and iOS. All components are made with [`react-native-unistyles`](https://www.unistyl.es/).

### Configure paths aliases
<Tabs groupId='framework' persist items={['React Native', 'Expo']}>

```js tab="React Native"
// tsconfig.json
{
  "compilerOptions": {
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}

// babel.config.js
module.exports = {
  plugins: [
    [
      'module-resolver',
      {
        alias: {
          '@': './src',
        },
      },
    ],
  ],
};
```

<Tabs groupId='framework' persist items={['React Native', 'Expo']}>
```tsx tab="React Native" title="App.tsx"
import { UnistylesProvider } from 'react-native-unistyles';

import '@/styles/unistyles';

export default function App() {
  return (
    <UnistylesProvider>
      {/* Your app */}
    </UnistylesProvider>
  )
}
```

```tsx tab="Expo" title="app/_layout.tsx"
import { UnistylesProvider } from 'react-native-unistyles';

import '@/styles/unistyles';

export default function RootLayout() {
  return (
    <UnistylesProvider>
      {/* Your app */}
    </UnistylesProvider>
  );
}
```

## Essential Components, Hooks and Utilities
To make the most out of this library, we recommend familiarizing yourself with the following components, hooks, and utilities. These are foundational and are used frequently across various components in the library:

- [**Text**](/docs/components/text): A versatile text component designed for consistent typography.
- [**Icon**](/docs/components/icon): A customizable icon component for adding icons to your UI.
- [**Slot**](/docs/utils/slot): A utility for flexible component composition.
- [**useControllableState**](/docs/hooks/useControllableState): A hook for managing controllable and uncontrollable states seamlessly.
