## Introduction

**Echo UI** is a collection of pre-built, high-quality React Native components designed to accelerate your mobile app development. Components are designed to deliver smooth and efficient performance on both Android and iOS. All components are made with [`react-native-unistyles`](https://www.unistyl.es/).

<Callout type='warn' title="Development preview">
Echo UI is currently in development and may subject to change. If you find any bugs or have any suggestions, please [open an issue](https://github.com/sarabjit20s/echo-ui/issues).
</Callout>

### Configure paths aliases
<Tabs groupId='framework' persist items={['React Native', 'Expo']}>

```js tab="React Native"
// tsconfig.json
{
  "compilerOptions": {
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}

// babel.config.js
module.exports = {
  plugins: [
    [
      'module-resolver',

          '@': './src',
        },
      },
    ],
  ],
};
```

2. Import `unistyles.ts` in the root of your project and wrap your app with `UnistylesProvider` component. For example:

<Tabs groupId='framework' persist items={['React Native', 'Expo']}>
```tsx tab="React Native" title="App.tsx"
import { UnistylesProvider } from 'react-native-unistyles';

import '@/styles/unistyles';

export default function App() {
  return (
    <UnistylesProvider>
      {/* Your app */}
    </UnistylesProvider>
  )
}
```

```tsx tab="Expo" title="app/_layout.tsx"
import { UnistylesProvider } from 'react-native-unistyles';

import '@/styles/unistyles';

export default function RootLayout() {
  return (
    <UnistylesProvider>
      {/* Your app */}
    </UnistylesProvider>
  );
}
```

<Step>
### Create a local development build
<Callout title="Dev client only">
Unistyles includes custom native code, which means **it does not support Expo Go**.
</Callout>

<Tabs groupId='framework' items={['React Native', 'Expo']}>
  <Tab value='React Native'>
    Run the following commands to build and run the app:
    ```bash
    # Android
    npx react-native run-android
    # iOS
    npx react-native run-ios
    ```
  </Tab>
  <Tab value='Expo'>
    1. Generate native directories
    ```bash
    npx expo prebuild --clean
    ```

    2. Run the following commands to build and run the app:
    ```bash
    # Android
    npx expo run:android
    # iOS
    npx expo run:ios
    ```
  </Tab>
</Tabs>
</Step>

These are some foundational components and hooks, which are used frequently across various components.

- [**Icon**](/docs/components/icon): A customizable icon component for adding icons to your UI.
- [**useControllableState**](/docs/hooks/useControllableState): A hook for managing controllable and uncontrollable states seamlessly.
